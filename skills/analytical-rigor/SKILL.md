# 分析严谨性

好的分析不是找更多支持自己的证据，而是主动寻找能推翻自己的证据。

## 触发时机

- `stock-report` Step 2 分析框架完成后，进入 PK 评分前
- 对任何超过 3 年的趋势进行外推时
- 某个结论显得过于确定，没有明显不确定性时
- 使用同行对比数据时

---

## 五项强制检查

### 检查 1 — 幸存者偏误扫描

A 股最常见、危害最深的统计陷阱，优先执行。

**核心问题**：我的参照系里有没有已经消失的案例？

常见形式：
- "这个行业平均 PE 高" → 亏损企业已被 ST 或退市，留下的都是质优的
- "同类公司平均增速 X%" → 增速低的已被并购或淘汰，不在样本里
- "历史上这类标的都涨了" → 跌到摘牌的不在你看到的历史记录里

**修正动作**：主动搜索"[行业] 退市 ST 案例 [近5年]"，将失败案例纳入参照系，
重新估算基准概率。

---

### 检查 2 — 近期外推限制

用过去 2-3 年的表现推断未来是最常见的过拟合。

**强制标注规则**：
- 基于 < 3 个数据点的趋势 → 标注「数据点不足，不构成趋势」
- 基于 3-5 个数据点 → 标注「短期趋势，需验证驱动因素的持续性」
- 基于 5 年以上 → 可作为趋势引用，但仍需说明假设

**外推前必须回答**：这个增速/下滑的驱动因素是结构性的，还是周期性的？
- 结构性：可以适度外推（但仍有上限）
- 周期性：严禁线性外推

---

### 检查 3 — 敏感性分析（只找最关键的一个）

不是列出所有变量，而是找到结论最依赖的那一个假设。

**操作步骤**：
1. 列出结论的 3-5 个核心假设
2. 对每个假设问：「如果这个假设错了 30%，结论会怎样？」
3. 找到那个「假设一旦错误，结论完全翻转」的关键变量
4. 在结论中明确标注：**本结论的关键假设是 [X]，如果 [X] 不成立，结论反转**

示例：
```
美年健康分析的关键假设排序
→ 第1位：营收能否在 Q2-Q3 企稳（如果继续下滑 10%+，结论从中性变为回避）
→ 第2位：AI 收入增速（即使降至 30%，对整体结论影响有限）
→ 第3位：应收账款（尾部风险，非基准情景）

核心结论只依赖第1位假设，盯住这一个指标比分析其余所有因素更重要
```

---

### 检查 4 — 置信度三档标注

每个结论必须标注属于哪档，不同档位使用不同表达方式：

| 档位 | 标准 | 允许的表达方式 |
|------|------|--------------|
| **数据证实** | ≥2 个 L1/L2 来源直接支持 | 「根据 XX 年报，X 为 Y」 |
| **合理推断** | 基于数据的逻辑推导，但无直接数据 | 「基于 X 趋势，Y 可能……」 |
| **主观判断** | 分析师判断，数据支撑薄弱 | 「我们倾向认为，但存在较大不确定性……」 |

**禁止**：用「数据证实」级别的确定语气表达「主观判断」档的内容。
**禁止**：结论中出现"一定"、"必然"、"肯定"等词，除非是引用 L1 法律事实。

---

### 检查 5 — 反向证据搜索

在结论形成后，主动搜索能推翻它的证据。

**正确的问法**：
- ❌「有没有更多数据支持我的结论？」
- ✅「什么数据会让我改变结论？去搜索这些数据。」

**操作**：明确写出"如果 [X] 为真，结论反转"，然后搜索 [X] 是否存在证据。

如果搜索后找不到任何反向证据：这不代表结论强，代表你搜索方向有问题。

---

## 与 `council` 的协作

当出现以下情况时，升级至 `council` 的 Skeptic 角色处理：

- 检查 3 发现结论极度依赖单一假设，且该假设本身难以验证
- 检查 5 发现重大反向证据，但与正向证据难以调和
- 置信度大量落在「主观判断」档，但结论仍然确定

`council` 的 Skeptic 是一个看不到你分析过程的独立 subagent，
它直接挑战你的前提假设，是这个流水线的最后一道防线。

---

## 输出前检查

- [ ] 幸存者偏误：检索了失败案例，校准了基准概率
- [ ] 外推：所有趋势预测标注了时间窗口和驱动因素性质
- [ ] 敏感性：识别并标注了结论最依赖的单一关键假设
- [ ] 置信度：每个核心结论都标注了所属档位
- [ ] 反向证据：主动搜索过能推翻结论的数据
